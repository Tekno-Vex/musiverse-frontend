# MusiVerse Frontend

React frontend for MusiVerse - An AI/ML-powered Spotify music companion.

## Tech Stack

- React 19.2.3
- Vite
- React Router
- Axios
- Tailwind CSS

## Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Make sure backend is running on `http://127.0.0.1:8080`
4. Start dev server: `npm run dev`
5. Open `http://127.0.0.1:5173/`

## Features

- Spotify OAuth login
- Protected routes
- User dashboard
- Responsive design
